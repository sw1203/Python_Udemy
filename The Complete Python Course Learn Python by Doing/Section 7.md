## Database (SQLite)

* python에서 SQLite를 이용하려면 sqlite3를 import
* database와 연결하려면 sqlite3.connect({db})를 이용
* .cursor()는 SQL qurey를 실행시킬수 있게 해준다.
* .excute({SQL QUERY})를 통해 sql문 실행
* .commit()을 통해 db에 변경사항 반영

## Cursor
